# 十神换算（a/b/c定义，a固定左边不动）

## 定义

- **a** = 属相地支五行
- **b** = 流日天干五行
- **c** = 流日地支五行
- 十神永远描述的是 **b/c 对 a 的关系**

## 十神查表

| b/c 对 a 的关系 | b/c阳 | b/c阴 |
|----------------|-------|-------|
| a克b/c | **偏财** | **正财** |
| b/c克a | **七杀** | **正官** |
| b/c生a（看a阴阳）| **印星**（a阳）| **枭神**（a阴）|
| a生b/c（看b/c阴阳）| **食神**（b/c阳）| **伤官**（b/c阴）|
| a同b/c | **比肩**（阴阳同）| **劫财**（阴阳异）|

# 计算步骤

1. 确认a（属相→地支→五行阴阳）
2. 确认b（流日天干→五行阴阳）
3. 确认c（流日地支→五行阴阳）
4. 用生克表判断b/c对a是什么关系
5. 代入十神表得出结果

**a永远固定在左边不动，十神描述的是b/c对a是什么。**
