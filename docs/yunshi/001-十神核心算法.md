# 开阳运势播报 · 十神核心算法

> 天枢 & 天尊 共创 · 2026-05-04
> 用途：九信阁 C 端用户运势播报

---

## 一、核心原则

**a = 用户属相的地支五行（永远固定在左边）**
**b = 流日天干五行**
**c = 流日地支五行**

运势的本质：流年/流月/流日的天干地支（b/c）对"我"（用户属相a）的生克比和关系，转化为十神，从而解读该时段的运势走向。

十神永远描述的是 **b/c 对 a 的关系**。

---

## 二、十神定义（b/c 对 a 的关系）

| b/c 对 a 的关系 | 十神 | 含义 |
|----------------|------|------|
| **a克b/c** | 财星 | 财运、物质、掌控、男命妻星 |
| **b/c克a** | 官杀 | 压力、责任、考验、权力、管控 |
| **b/c生a** | 印星/枭神 | 贵人、庇护、学习、证书、靠山 |
| **a生b/c** | 食伤 | 付出、表达、创作、投资、泄气 |
| **a同b/c** | 比肩/劫财 | 竞争、兄弟姐妹、同事、破耗、争斗 |

---

## 三、阴阳与十神细分

在同一关系下，阴阳不同，十神名称不同：

| b/c 对 a 的关系 | b/c阳 | b/c阴 |
|----------------|-------|-------|
| a克b/c | **偏财** | **正财** |
| b/c克a | **七杀** | **正官** |
| b/c生a（看a阴阳）| **印星**（a阳）| **枭神**（a阴）|
| a生b/c（看b/c阴阳）| **食神**（b/c阳）| **伤官**（b/c阴）|
| a同b/c | **比肩**（阴阳同）| **劫财**（阴阳异）|

---

## 四、输入与输出

### 输入
- `shengxiao`：用户属相（如"子"=鼠）
- `year_gan` / `year_zhi`：流年干支
- `month_gan` / `month_zhi`：流月干支
- `day_gan` / `day_zhi`：流日干支

### 输出
- `year_shishen`：流年十神
- `month_shishen`：流月十神
- `day_shishen`：流日十神
- `year_relation`：流年天干/地支与属相的五行关系
- `month_relation`：流月天干/地支与属相的五行关系
- `day_relation`：流日天干/地支与属相的五行关系

## 五、算法步骤

### Step 1：定义基础数据

**天干属性**
```
甲→阳木  乙→阴木  丙→阳火  丁→阴火
戊→阳土  己→阴土  庚→阳金  辛→阴金
壬→阳水  癸→阴水
```

**地支属性**
```
子→阳水  丑→阴土  寅→阳木  卯→阴木
辰→阳土  巳→阴火  午→阳火  未→阴土
申→阳金  酉→阴金  戌→阳土  亥→阴水
```

**五行生克（克方→被克方）**
```
相生：木→火→土→金→水→木
相克：木→土→水→火→金→木
```

### Step 2：提取五行与阴阳

- a（属相地支）→ 五行 + 阴阳
- b（流日天干）→ 五行 + 阴阳
- c（流日地支）→ 五行 + 阴阳

### Step 3：判断五行关系

用生克表判断 a vs b、a vs c 的关系：

- a克b/c → 查偏财/正财
- b/c克a → 查七杀/正官
- b/c生a → 查印星/枭神（看a的阴阳）
- a生b/c → 查食神/伤官（看b/c的阴阳）
- a同b/c → 查比肩/劫财（看阴阳是否相同）

**注意：a永远固定在左边不动，十神描述的是b/c对a是什么。**

### Step 4：查十神表

根据阴阳判断具体十神名称。

## 六、示例计算（a=卯阴木）

**用户属相：卯（木，阴）**

**流日：己卯日**
- a=卯阴木，b=己阴土，c=寅阳木
- a vs b：木克土（a克b），阴阳同 → **偏财**
- a vs c：木同木，阴阳异 → **劫财**

**流日：庚辰日**
- a=卯阴木，b=庚阳金，c=辰阳土
- a vs b：金克木（b克a），阴阳异 → **正官**
- a vs c：木克土（a克c），阴阳异 → **正财**

**流日：辛巳日**
- a=卯阴木，b=辛阴金，c=巳阴火
- a vs b：金克木（b克a），阴阳同 → **七杀**
- a vs c：木生火（a生c），阴阳同 → **食神**
